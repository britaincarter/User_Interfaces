About

This website allows you can to control multiple videos through simple mouse gestures. Try it on the canvas to the right!

Demo

In the square to the right, you can draw the commands to manipulate videos which are located in the column to the left. New and confused? Read the Quick Guide below to help get started!


Make strokes on this canvas by clicking and holding your left mouse button.

The <canvas> element is not supported by this browser.

Videos

Quick Guide

  1. To manipulate a video there must be a video to manipulate. Try adding a video! (Draw the command on the canvas named "Create New Video")
  2. To manipulate specific videos use the select gesture in the canvas, then click on the videos you want to select!
  3. Now you can manipulate a specific videos attributes by using the gestures on the canvas!
  4. Done manipulating a specific video? You can de-select a video by using the select gesture and clicking on the videos you want to de-select!
  5. If you want to manipulate all the videos at once use a command for all videos.
  6. You have all the tools you need. Good luck and have fun!
